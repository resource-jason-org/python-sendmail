### python-sendmail

- python-sendmail 是一个自动发邮件的程序

- 如何配置

```
配置文件可以将 config.ini_example 重命名为 config.ini
修改为自己的配置
```

- 如何启动

```
python sendmail.py
```

- 邮件列表

```
emailList.xml 文件中存放需要发送的邮件列表
格式例如:
test@gmail.com,
test2.gmail.com,
test3.gmail.com
```
- 邮件内容

```
mailcontent.html 保存你要发送的邮件内容,支持html格式
```

- log

```
sendmail.log
```
